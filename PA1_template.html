Create a new dataset that is equal to the original dataset but with the missing data filled in. We are going to use the means of the intervals to fill in the data.

data.filled <- data
index.na <- is.na(data.filled$steps)
# Create a factor variable out of the interval numeric value. 
data.filled$interval <- factor(data.filled$interval)
